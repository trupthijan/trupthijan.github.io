Fundamentals of Software Engineering

Course Code: A54030

Offered to: B.Tech AI & AIML- 2nd Year (3rd Semester)

Instructor: Dr. M. Trupthi

Topics Covered

  1. Introduction to Software Engineering
  2. Software Development Life Cycle Models
  3. Requirement Analysis and Design
  4. UML Diagrams and Architecture Design
  5. Agile Methodologies and Scrum
  6. Software Testing and Maintenance
  7. Project Estimation and Planning
  8. Tools: GitHub, Jira, Draw.io

Course Objectives

  • To introduce the basics of software engineering and types of software development projects in software industry
  • To classify various Software Life Cycle Models
  • To identify and select suitable Software Project Management Processes
  • To list the steps or phases of Software Development Process and related artifacts
  • To use Agile Software Development practices and DevOps

Course Outcomes

  1. After Cpmpletion of the course students will be able to:
  2. Understand the emergence of software engineering and types of software development projects
  3. Infer merits and demerits of software life cycle models and selecting appropriate model for a project
  4. Apply the software project management practices and techniques essential for successful completion of a project
  5. Analyze the steps or phases involved in software development processes and its related artifacts
  6. Apply Agile Software Development practices and DevOps

Assessment

  • Assignments: 20%
  • Mid-term Exam: 30%
  • Final Exam: 50%
