Unit I

  • Trees: Tree definition, Tree applications, Types of trees- Binary tree, Complete Binary tree, Full Binary tree, Extended Binary Tree, Threaded Binary Tree, Representation of binary tree using arrays and linked lists (advantages and disadvantages), Binary tree traversals, Algorithms for tree traversals. Heaps: Introduction, Types of heaps- Min binary heap and Max binary heap.

Unit II

  • Binary Search trees- Binary search tree definition, Representation and creation of Binary Search Tree (BST), Algorithm for inserting, deleting and searching in BST, AVL trees: Representation and advantages of AVL Tree, Algorithms on AVL Tree operations- insertion, rotation and deletion.

Unit III

  • Graphs- Graph definition, Applications of graph, Representation of graphs- Adjacency matrix and Linked List. Graph Traversals: Breadth First Search and Depth First Search with algorithms. Differences between DFS and BFS.

Unit IV

  • Spanning trees: Definition and Properties of Spanning tree, Minimum Cost Spanning Tree, Algorithms for minimum cost- Prim’s algorithm, Kruskal’s algorithm, Dijkstra’s algorithm. B-trees: Definition and advantages of B-tree, B-tree of Order M, B-tree Operations- insertion and searching

Unit V

  • Hashing: Hashing definition, Hash function, Collision in hashing, Collision resolution techniques- Closed Addressing (Separate chaining), Open Addressing (Linear probing, Quadratic probing, Double hashing), Rehashing, Extensible hashing.

Text Books & References

  • Data Structures, Seymour Lipschutz, Schaum’s Outlines, Tata McGraw-Hill, Special Second Edition.
  • Aaron M. Tenenbaum, Yedidyah Langsam and Moshe J. Augenstein, Data Structures Using C and C++, PHI Learning Private Limited, Delhi India.
  • Data Structure Using C, A.K. Sharma, Pearson Education India.
